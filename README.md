# Order Delivery CLI

This is a simple Command-Line Interface (CLI) application written in C++ to manage basic order and delivery operations. It allows users to enter order details, process deliveries, and display stored information using a clean, menu-driven console interface.

## Features
- Add new orders  
- View existing orders  
- Update or process delivery details  
- Console-based user interaction  
- Beginner-friendly and easy to run  

## Tech Stack
- C++  
- Console-based interface
